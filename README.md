# MISIS-PE

**MISIS-PE** — это сервис для регистрации на занятия физкультурой в университете МИСиС.


## Основной функционал

- Регистрация новых пользователей и авторизация в личномм кабинете
- Запись на занятия
- Получение уведомлений о грядущих мероприятиях

## Стек технологий

### Frontend

- **JS React**

### Backend

- **FastAPI**
- **SQLAlchemy**
- **PostgreSQL**

### DevOps

- **Docker**
- **Docker Compose**
- **Nginx**

  
### Тестирование
